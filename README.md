    <tr>
        <td>ShowTheCall</td>
        <td>

        <td>SwitchOption</td>
        <td>
            Es sollte ein Programm geschrieben werden welches über verschiedene Kommandozeilenargumente

            Sinn der Übung war es den Syntax des <code>switch...case</code> Statements zu erlernen.
        </td>
    </tr>
    <tr>
        <td>Texteingaben verarbeiten</td>
        <td>
            Es sollte ein Programm geschrieben werden welches eine Texteingabe entgegennimmt und verarbeitet.
